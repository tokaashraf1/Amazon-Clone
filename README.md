# Amazon-Clone ( E-commerce Application )
## Intoduction
### A professional React application designed for online shopping, offering customers a seamless and secure shopping experience
### The system provides:
1- Authentication - making sure that the customer securely log in to access the system

2- View Products – Browse available products with the ability to search them and filter them based on brands A and prices.

3- Add to Cart – Select products and add them to the cart for purchase.

4- Add to WishList – Choosing products to your favourite list to check and order them later.

5- View Product details – Browse all information about the product such as reviews and descriptions  .

## Technical skills:
- <b>Reactjs</b> : for building user interfaces with components.
- <b>TailwindCSS</b>: Utility-first CSS framework for styling.
- <b>Axios</b>: HTTP client for API requests.
- <b>React Router</b>: for Navigation and routing in React apps.
- <b>Context API</b>: State management to share data globally across components.
